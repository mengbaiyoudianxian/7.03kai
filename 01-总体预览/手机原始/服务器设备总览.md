## 各设备详细介绍

### 1. 母体机 (47.83.2.188)
- **角色**: 整个 MBclaw 的核心服务器
- **运行内容**:
  - FastAPI 后端 (uvicorn :8000, nginx代理到80)
  - 管理面板 (admin panel v5.5)
  - Claude Code (你正在用的AI助手)
  - 所有管理 API (40+ 端点)
- **关键文件路径**:
  - `/opt/mbclaw/app/` — 运行中的后端代码
  - `/var/lib/mbclaw/` — 数据目录 (用户/keys/心跳/heartbeat_logs)
  - `/etc/nginx/conf.d/mbclaw.conf` — nginx配置
- **服务**: `systemctl restart mbclaw`
